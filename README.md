# Сервис ValutaTrade Hub

Проект реализует сервис хранения и обмена валютами.
Данные курсов берутся из публичных API [CoinGecko](www.coingecko.com) и [ExchangeRate](exchangerate-api.com).
Пользователь может смотреть обменные курсы валют, покупать, продавать и хранить их в кошельке.

Поддерживаемые валюты: `USD`, `EUR`, `RUB`, `GBP`, `BTC`, `SOL`, `ETH`.

## Требования к установке

* **Python** от 3.13;
* **Poetry** от 2.0.0 до 3.0.0;
* **Ruff** от 0.14.7;
* **Requests** от 2.32.5 до 3.0.0;
* **Prettytable** от 3.17.0 до 4.0.0

и/или

* **GNU Make**.

## Установка

Для начала зарегистрируйтесь на сайте [exchangerate-api.com](exchangerate-api.com) и получите ключ к API.

---

Распакуйте проект в удобную директорию. После выполнения требований введите в установленной директории:
```
make install
```
или
```
poetry install
```

---

Задайте переменную среды:
```
export EXCHANGERATE_API_KEY="api_key"
```
где `"api_key"` - это ваш ключ к API.

---

После успешной установки можно начать пользоваться:
```
make run
```
или
```
poetry run valutatrade
```

## Использование

Вызов команд:
`<command>` `<--argument1>` `<input>` `<--argument2>` `<input>` ...

Список команд:

- `register` - регистрация в системе.

Обязательные аргументы:

`--username` - имя пользователя. Не может быть пустым.

`--password` - пароль. Должен содержать минимум 4 символа.

- `login` - авторизация в системе.

Обязательные аргументы:

`--username` - имя пользователя.

`--password` - пароль.

- `show-portfolio` - информация о ваших кошельках. Требует авторизации.

Необязательные аргументы:

`--base` - код валюты (например: USD), по которой учитывать общий баланс кошельков.

- `buy` - купить валюту. Требует авторизации.

Обязательные аргументы:

`--currency` - код валюты (например: USD), которую хотите купить.

`--amount` - объем покупки в штуках (например: 123.45). Неотрицательное число.

- `sell` - продать валюту. Требует авторизации.

Обязательные аргументы:

`--currency` - код валюты (например: USD), которую хотите продать.

`--amount` - объем продажи в штуках (например: 123.45). Неотрицательное число, нужно иметь необходимые средства в кошельке.

- `get-rate` - получить курс одной валюты к другой.

Обязательные аргументы:

`--from` - код исходный валюты (например: EUR).

`--to` - код целевой валюты (например: USD).

- `show-rates`` - получить курсы валют по фильтру.

Необязательные аргументы:

`--currency` - код исходной валюты (например: EUR).

`--base` - код базовой валюты (например: USD).

`--top` - число (например: 5) верхних строк, которые хотите вывести на экран.

- `update-rates` - обновить обменные курсы валют.
Курсы валют считаются устаревшими спустя 6 минут.
Они также обновляются, если вы покупаете/продаете валюты или просматриваете кошелек.

Необязательные аргументы:

`--source` - один из сервисов API (coingecko или exchangerate). Если не указывать, будут задействованы оба сервиса.

- `exit` - выход.
